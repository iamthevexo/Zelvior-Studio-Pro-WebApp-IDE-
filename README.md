# Zelvior Studio Pro

**Zelvior Studio Pro** is a self-made Web App IDE developed by Faizan.  
This project allows users to create, edit, and test HTML, CSS, and JavaScript code directly in the browser.  

---

## Features

- Full support for **HTML, CSS, JavaScript** editing
- Live preview of your code
- JSON data integration for dynamic content
- Clean and modern **glassmorphism UI**
- Lightweight and fast
- Designed for personal use (private logic, not open source)

---

## Usage

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Start editing HTML, CSS, or JS in the IDE interface.
4. Add or manage JSON data through `data.json`.
